Creacion de formularios

Utilizar < form> < /form> para crear el formulario

Para indicar donde el formulario sera enviado, utilizar el atributo: action="Aqui la url donde sera enviado". Ejemplo:

< form action="/submit/url">
< /form>

Dentro del formulario pueden crearse inputs:

Input tag y atributos:

Atributos de input:

  • type="text"
  • name="firstname"
  • placeholder="some text"
  • required

Utilizar < label> para asociar el texto del input con el propio input:

< label> Este es el label del input
< input>
< /label>
